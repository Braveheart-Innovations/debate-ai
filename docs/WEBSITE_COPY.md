## ADDITIONAL NOTES FOR SQUARESPACE

### Recommended Image Sizes
- Hero images: 1920x1080px minimum
- App screenshots: Use actual device frames (iPhone/Android mockups)
- Feature icons: 128x128px or use Squarespace's built-in icons

### Color Suggestions
Consider aligning the site's accent color with Symposium AI's branding for visual consistency across your product and company site.

### Typography
Keep it clean and readable. Squarespace's default system fonts work well, or consider a modern sans-serif like Inter, Open Sans, or Montserrat.
